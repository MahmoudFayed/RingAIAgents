## كيفية تشغيل المثال

1. افتح النافذة الرئيسية للتطبيق
2. استكشف الفرق المختلفة وأعضائها
3. راقب المهام وتقدمها
4. جرب ميزات الدردشة مع العملاء
5. استخدم لوحة المراقبة لتتبع الأداء
